WHAT ARE SVG FILES, AND HOW DO THEY WORK?


SVG is a very versatile file format made up of vector graphic elements. They do not use pixels, but rather mathematical curves, paths, gradients, boxes, strokes, text, etc. This not only allows for a much smaller file size, but it lets you scale your graphics to any resolution you desire. SVGs are used as source files for logos, t-shirt designs, bed sheet designs, digital templates, web icons, billboards, window stickers, car stickers, etc. These files are not designed for complex photos, or other images with a lot of detail and texture. Trying to convert a PNG or JPG into an SVG might work if it's simple enough, but if it's a large, complex image with a lot of detail, it can get quite resource-intensive to convert to SVG. SVG files are not to be treated the same way as PNGs, but rather as "project files", or "source files", where you can pick each vector element apart, reuse it for other projects, and export it separately into PDFs, PNGs, etc. The best way to create and open SVG files is either through paid programs like Adobe Illustrator, or free and open source alternatives, such as Inkscape. I personally use Inkscape, because I'm not coughing up $60 per month for only a handful of programs from the Adobe Creative Cloud.
